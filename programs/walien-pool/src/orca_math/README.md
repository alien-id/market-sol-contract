# orca_math

This math module is get from the Orca Whirlpools implementation at commit
[`8efab65f31e09ad845f2c0af8d12548e3a81cdb0`](https://github.com/orca-so/whirlpools/tree/8efab65f31e09ad845f2c0af8d12548e3a81cdb0/programs/whirlpool/src/math).
